\subsection{Supplementary Text}\label{supplementary-text}

Use subheadings as needed (for example, ``Study site'', ``Analytical
pipeline'', ``Sensitivity analyses''). Keep paragraphs left-aligned with
line breaks between them.

\subsection{Figures}\label{figures}

Place each figure on its own page above the caption.

\subsubsection{Fig. S1. Descriptive
title}\label{fig.-s1.-descriptive-title}

Begin the caption with the figure identifier, followed by a concise
description of panels and symbols.

\subsection{Tables}\label{tables}

Each table should appear on a separate page. Keep wide tables as
separate CSV/Excel attachments if they exceed the page width.

\subsubsection{Table S1. Summary
statistics}\label{table-s1.-summary-statistics}

Provide a brief description of the data shown and any abbreviations
used.

\subsection{Captions for Movies}\label{captions-for-movies}

List media files included outside the PDF. Example: \textbf{Movie S1.} A
30-second time-lapse showing particle tracking (MP4, 25 MB).

\subsection{Captions for Data}\label{captions-for-data}

Describe supplemental data files or archives. Example: \textbf{Data S1.}
CSV containing raw measurements for Figures 2--3.

\subsection{References}\label{references}

Cite references in continuation of the main-text numbering, italicized
in parentheses per Science Advances style.
